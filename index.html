GitHub 연동 설정

데이터를 GitHub Gist에 저장해서 어디서든 동기화합니다. 1회만 설정하면 됩니다.

Step 1
Personal Access Token 발급

GitHub → Settings → Developer settings
→ Personal access tokens → Tokens (classic)
→ Generate new token → gist 권한만 체크 → 생성

생성된 토큰(ghp_...)을 복사하세요.

Step 2
토큰 입력

토큰은 이 기기의 브라우저에만 저장됩니다.

토큰이 올바르지 않거나 gist 권한이 없습니다.
